## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The site will open at `http://localhost:3000`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Customization

Edit `index.html` to add:
- Josh's actual photo (replace the hero image URL)
- Burial dates and times
- Location details
- Eulogy text
- Contact information
- Personal photos in the gallery section

## Structure

- `index.html` - Main HTML structure
- `src/style.css` - All styling and animations
- `src/main.js` - JavaScript for interactions and animations
