# Images Folder

Place your photos here:

## Hero Image (Main Profile Photo)
- **File name:** `hero.jpg` (or `hero.png`)
- **Recommended size:** 400x400px or larger (square format works best)
- This will be displayed as the main profile photo at the top of the page

## Gallery Images (Memory Photos)
- **File names:** `memory1.jpg`, `memory2.jpg`, `memory3.jpg`, `memory4.jpg` (or `.png`)
- **Recommended size:** 800x600px or larger
- You can add more images by adding more gallery-item divs in index.html
- Supported formats: JPG, PNG, WebP

## Notes:
- You can use any number of photos - just update the HTML accordingly
- If you have fewer than 4 photos, remove the extra gallery-item divs from index.html
- If you have more photos, duplicate the gallery-item div structure and update the image paths
